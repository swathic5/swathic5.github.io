Once Software Engineer, Now Computer Vision grad

I am a Master's student in Carnegie Mellon University studying Computer Vision. My passion for Computer Vision started when I was introduced to classical Image Processing techniques during my undergraduate sophomore. Starting from then, I have constantly involved myself in many pet projects and real projects!

I did my undergraduate junior year internship at Texas Instruments, Bangalore, India. It was the first true transition into pure Computer Vision. I got introduced to multi-view geometry by Zisserman which is one book that I cherish! I got to develop an algorithm to segment the road in street scenes. I used homography with modification which I published and patented.

I have worked at Texas Instruments, Bangalore, India for 3 awesome years. I had the best oppotunity to work on different levels of embedded vision application development. I have worked on OpenCL based algorithm acceleration, assembly level code optimization, dataset creation, machine learning, deep learning, OpenCV porting to TI BIOS.

Intelligent Inpainting

role: author

Intelligent Inpainting is a Computer Vision application developed in C++. It takes a single image and a user click. The click is intented to select a person on the input image. Then the application intelligently crops that person out of the image by filling with the background.
